LogPro - Developed By Jonathan Strickland

About LogPro:
LogPro is a community service tracking/management application designed for school systems.

** Please note: this program is optimized for Mac OS **

How to begin:

Step 1) Launch LogPro.jar.

Step 2) Either log in to the admin account, or create a student or instructor account.
